# Dank Clothing API

Backend REST API for the Dank Clothing application. This API is built with Laravel and provides endpoints for managing products, users, orders, authentication, and other application resources.

## Frontend

The Vue.js frontend for this project can be found here:

- https://github.com/AJSolis/dank-clothing

## Prerequisites

Before running this project, install the following software:

### Required Software

- PHP 8.x or newer
- Composer
- MySQL Server
- Visual Studio Code (recommended)
- MySQL Workbench
- Postman (for API testing)

### Downloads

- PHP: https://www.php.net/downloads
- Composer: https://getcomposer.org/download
- MySQL Server: https://dev.mysql.com/downloads/mysql/
- Visual Studio Code: https://code.visualstudio.com/
- MySQL Workbench: https://dev.mysql.com/downloads/workbench/
- Postman: https://www.postman.com/downloads/
